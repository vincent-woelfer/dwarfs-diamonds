# ⛏️ Dwarfs & Diamonds

A 2D colony simulation game built with Godot 4, featuring dwarfs that mine, build, expand and survive underground.

**Work in Progress**

![](txt/screenshots/screenshot_1.png)

## About

Dwarf Colony is a centralized multi-agent strategy game where you oversee a colony of dwarfs carving out a home in the depths of the earth. Assign tasks, manage resources, and watch your colony grow.

Features:
- **Task Scheduling & Distribution** — a centralized system assigns jobs (mining, hauling, building) to available dwarfs based on priority and proximity
- **Dynamic Pathfinding** — agents navigate a fully destructible tile-based world in real time
- **Colony Simulation** — dwarfs act as individual agents responding to centralized orders

## Running the Project

1. Clone the repo
2. Open `project.godot` in Godot 4.6+
3. Hit **F5** to run

No external dependencies required.
